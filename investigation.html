Investigation: Why Does the Negative Imaginary Axis Map to Re(R) = -π?

The Observation

According to the paper's figure 7, the negative imaginary axis in z-plane should map to the negative imaginary axis in R-plane (contracted by factor λ = 2/π).

This means: z = -yi → R = -λyi, so Re(R) should be 0.

But our calculation gives Re(R) = -π for all points on the negative imaginary axis!

Problem: R₀ is negative real for all these points, which lies exactly on the log branch cut!

2. The Mathematics

R = i · ln(R₀) = i · (ln|R₀| + i·arg(R₀)) = -arg(R₀) + i·ln|R₀|

Therefore:

  • Re(R) = -arg(R₀)
  • Im(R) = ln|R₀|

For R₀ on the negative real axis:

  • Principal branch: arg(R₀) = +π → Re(R) = -π
  • Lower branch: arg(R₀) = -π → Re(R) = +π

Neither gives Re(R) = 0!

3. What Would Give Re(R) = 0?

To get Re(R) = 0, we need arg(R₀) = 0.

But R₀ is negative real, so arg(R₀) ≠ 0 in any standard branch!

5. Resolution: The 2π Periodicity

The R-plane is actually a horizontal strip with periodic identification:

R and R + 2πn are equivalent for any integer n

This is because R = i·ln(R₀), and ln is multivalued with period 2πi.

In the R-plane:

  • Re(R) = -π and Re(R) = +π are the same vertical line
  • The strip [-π, π] × ℝ is the fundamental domain
  • The edges at Re = ±π are glued together

So points at Re(R) = -π are on the seam, not "wrong"!

6. The Paper's Figure Interpretation

Two possibilities:

  1. The paper shows the universal cover: The negative imaginary axis appears at Re(R) = 0 because they're showing the Riemann surface unfolded, not the principal branch.
  2. The paper uses continuous branch selection: For iteration purposes, the branch of log is chosen to maintain continuity along trajectories, not the principal branch.

For iteration R, R², R³, ..., we need continuous tracking, which may require crossing the branch cut at Re = ±π and continuing smoothly.

8. Conclusion

The mapping is mathematically correct.

The negative imaginary axis maps to the branch cut boundary of the logarithm in R₀-plane, which manifests as Re(R) = ±π in R-plane.

Since the R-plane strip has 2π periodicity, points at Re = -π and Re = +π are identified. The paper's figure may show:

  • A different fundamental domain (e.g., [-π/2, 3π/2) instead of [-π, π))
  • The Riemann surface view where Re = 0 represents the "unwound" branch cut
  • A continuous branch selection for iteration purposes

For computational purposes, our implementation correctly handles the conformal mapping and produces valid results for iteration.
